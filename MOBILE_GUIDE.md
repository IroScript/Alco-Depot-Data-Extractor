# 📱 Alco Pharma Dashboard - Mobile Testing & Deployment Guide

## 🌐 ১. মোবাইল থেকে লোকালহোস্টে টেস্ট করার নির্দেশিকা (Local Testing via Mobile)

আপনার পিসিতে সার্ভার চালু থাকলে, আপনার মোবাইল একই ওয়াইফাই / মোবাইল হটস্পট নেটওয়ার্কে যুক্ত থাকলে সরাসরি মোবাইল দিয়ে লোকাল ড্যাশবোর্ড দেখা যাবে।

### ধাপসমূহ:
১. **পিসি ও মোবাইল একই নেটওয়ার্কে যুক্ত করুন:** 
   - পিসির ওয়াইফাই এবং মোবাইলের ওয়াইফাই একই রাউটার বা মোবাইলের হটস্পটের সাথে কানেক্টেড থাকতে হবে।

২. **পিসিতে লোকাল ড্যাশবোর্ড সার্ভার রান করুন:**
   - টার্মিনালে `Dashboard` ফোল্ডারে গিয়ে কমান্ড দিন:
     ```bash
     python -m http.server 8080 --bind 0.0.0.0
     ```

৩. **পিসির IP অ্যাড্রেস চেক করুন:**
   - টার্মিনালে `ipconfig` লিখে আপনার IPv4 Address দেখুন (যেমন: `10.196.246.16`)।

৪. **মোবাইলের ব্রাউজারে প্রবেশ করুন:**
   - মোবাইলের **Chrome** বা **Safari** ওপেন করে ব্রাউজারের URL বারে টাইপ করুন:
     ```
     http://<YOUR_PC_IP>:8080
     ```
     *(উদাহরণ: `http://10.196.246.16:8080`)*
     > ⚠️ **গুরুত্বপূর্ণ:** খেয়াল রাখবেন শুরুতে যেন **`http://`** থাকে (`https://` দিলে কাজ করবে না)।

## 💻 ২. পিসির ব্রাউজারে মোবাইল ভিউ (Mobile View Emulator) টেস্ট করা

১. ব্রাউজারে `http://127.0.0.1:8080` লিংকে প্রবেশ করুন।
২. কিবোর্ডে **`F12`** চাবুন (Developer Tools)।
৩. কিবোর্ডে **`Ctrl + Shift + M`** চাপুন (Mobile View Toggle)।
৪. ওপর থেকে যেকোনো মোবাইল ডিভাইস (iPhone/Android) সিলেক্ট করে মোবাইল ভিউ পরীক্ষা করুন।

---

## 🚀 ৩. PythonAnywhere (PA) লাইভ সার্ভার আপডেট নির্দেশিকা

গিটহাব থেকে সরাসরি ক্লোন করার পর অথবা লাইভ সার্ভার আপডেট করতে:
- আপনাকে আলাদা কোনো Tailwind CSS ইন্সটল বা বিল্ড করতে হবে না।
- শুধু **`Dashboard/index.html`** এবং **`Dashboard/js/script.js`** ফাইল দুটি আপডেট/রিপ্লেস করলেই ড্যাশবোর্ডের সমস্ত নতুন পপআপ, ফিল্টার ও সিঙ্ক ফিচার লাইভ হয়ে যাবে।

## 🔄 ৪. নতুন পিসিতে গিট ক্লোন (Git Clone) সংক্রান্ত বার্তা
আপনার রিপোজিটরিটি গিটহাবে পুরোপুরি নিরাপদ ও আপ-টু-ডেট আছে। নতুন কোনো কম্পিউটারে গিট ক্লোন করতে চাইলে নিচের কমান্ডটি দিন:
```bash
git clone https://github.com/IroScript/Alco-Depot-Data-Extractor.git
```
ক্লোন করার সাথে সাথেই সমস্ত সোর্স কোড ও ড্যাশবোর্ড এক ক্লিকে কাজ করবে।
