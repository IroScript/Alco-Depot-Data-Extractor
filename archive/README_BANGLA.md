# MPO ডেটা এক্সট্র্যাক্ট করার সম্পূর্ণ গাইড

## 🎯 আপনার সমস্যা:
- পুরোনো সফটওয়্যার থেকে ২০০০ MPO এর ডেটা ম্যানুয়ালি বের করতে হয়
- প্রতিটি MPO এর জন্য আলাদা কমান্ড দিতে হয়
- এতে অনেক সময় লাগে

## ✅ সমাধান: Python দিয়ে অটোমেশন (SQL Server লাগবে না!)

---

## 📋 পদক্ষেপ ১: Python ইনস্টল করুন

1. https://www.python.org/downloads/ থেকে Python ডাউনলোড করুন
2. ইনস্টল করার সময় **"Add Python to PATH"** চেকবক্স টিক দিন
3. ইনস্টল শেষে Command Prompt খুলে টাইপ করুন: `python --version`

---

## 📋 পদক্ষেপ ২: প্রয়োজনীয় লাইব্রেরি ইনস্টল করুন

Command Prompt খুলে এই কমান্ড চালান:

```cmd
pip install pyautogui pandas openpyxl pyperclip pillow
```

## 📋 পদক্ষেপ ৪: স্ক্রিপ্ট চালান

### পদ্ধতি A: GUI Automation (সবচেয়ে সহজ - আপনার জন্য বেস্ট)

1. আপনার পুরোনো সফটওয়্যার ওপেন করুন
2. যেখানে MPO আইডি টাইপ করতে হয় সেই স্ক্রিনে যান
3. Command Prompt খুলে টাইপ করুন:

```cmd
cd "c:\Users\Irak\Desktop\Barishal April Data"
python automate_mpo_extraction.py
```

4. MPO লিস্ট ফাইলের নাম দিন (যেমন: `mpo_list.xlsx`)
5. 10 সেকেন্ডের মধ্যে সফটওয়্যারের ইনপুট বক্সে কার্সর রাখুন
6. স্ক্রিপ্ট অটোমেটিক সব MPO এর ডেটা বের করবে!

**⚠️ গুরুত্বপূর্ণ:**
- স্ক্রিপ্ট চলার সময় মাউস বা কিবোর্ড টাচ করবেন না
- কম্পিউটার লক করবেন না
- অন্য কোনো উইন্ডো ওপেন করবেন না

## 🔧 কাস্টমাইজেশন

### যদি আপনার সফটওয়্যারে Enter এর বদলে অন্য কী চাপতে হয়:

`automate_mpo_extraction.py` ফাইল খুলে এই লাইন খুঁজুন:

```python
pyautogui.press('enter')
```

পরিবর্তন করুন:
- F5 চাপতে হলে: `pyautogui.press('f5')`
- Tab চাপতে হলে: `pyautogui.press('tab')`
- Ctrl+S চাপতে হলে: `pyautogui.hotkey('ctrl', 's')`

### যদি ডেটা লোড হতে বেশি সময় লাগে:

এই লাইন খুঁজুন:
```python
time.sleep(2)  # ডেটা লোড হওয়ার জন্য অপেক্ষা
```

`2` এর জায়গায় `5` বা `10` দিন (সেকেন্ডে)

## ❓ সমস্যা হলে

### Error: "No module named 'pyautogui'"
**সমাধান:** 
```cmd
pip install pyautogui
```

### Error: "File not found"
**সমাধান:** 
- ফাইলের পাথ ঠিক আছে কিনা চেক করুন
- ফাইলের নাম সঠিক আছে কিনা দেখুন

### স্ক্রিপ্ট ভুল জায়গায় ক্লিক করছে
**সমাধান:**
- স্ক্রিনের রেজোলিউশন চেক করুন
- সফটওয়্যার উইন্ডো ফুলস্ক্রিন করুন
- `time.sleep()` এর মান বাড়ান
